Attribute Macros

  • async_init
    Mark async function to be the program initialization method. Can be used together with async_main. Function init cannot be specified if this macro is used. If you need to specify init explicitly don’t use this macro.

    This is the procedural macro for your convenience. It marks the main async function to be the program entry point. Functions handle cannot be specified if this macro is used. If you need to specify handle explicitly don’t use this macro.
